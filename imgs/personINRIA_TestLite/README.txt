UPPER-BODY FRONTAL Test set
~~~~~~~~~~~~~~~~~~~~~~~~~~~

This package contains a subset of INRIA person dataset, http://pascal.inrialpes.fr/data/human/

From the INRIA test set, we have chosen the 145 images that contains only frontal views of persons, and which size suits the minimum one of the upper-body frontal detector (100x90).

The upper-body detector has been firstly used at:

Ferrari, V. , Marin-Jimenez, M. and Zisserman, A.
Progressive Search Space Reduction for Human Pose Estimation
Proceedings of the IEEE Conference on Computer Vision and Pattern Recognition (2008)
